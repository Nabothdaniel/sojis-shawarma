# BamzySMS Backend Deployment Instructions (cPanel)

This package is optimized for hosting the backend in the `public_html/api/` folder.

## 1. Upload to cPanel
1. Log in to cPanel and open **File Manager**.
2. Navigate to `public_html/`.
3. Upload `backend_ready.zip`.
4. Right-click the file and select **Extract**.
5. This will create a folder named `api/`.
6. (Cleanup) Delete `backend_ready.zip` from the server.

## 2. Database Setup
1. In cPanel, go to **MySQL Databases**.
2. Create a new database (e.g., `youruser_bamzysms`).
3. Create a new database user and set a strong password.
4. Add the user to the database and grant **ALL PRIVILEGES**.
5. Go to **phpMyAdmin**, select your new database, and click **Import**.
6. Choose the `database_production.sql` file included in this project root (locally) and upload it.

## 3. Configuration (.env)
1. In cPanel File Manager, open the `public_html/api/` folder.
2. If there is a `.env.example`, rename it to `.env`.
3. Edit `.env` and fill in your details:
   ```env
   DB_HOST=localhost
   DB_NAME=tiheogwi_bamzy_sms
   DB_USER=tiheogwi_bamzy_admin
   DB_PASS=tiheogwi_bamzy_admin
   SMSBOWER_API_KEY=your_actual_key_here
   CORS_ALLOWED_ORIGINS=https://yourdomain.com
   ```

## 4. Troubleshooting
- **404 Not Found**: Ensure `public_html/api/.htaccess` was extracted successfully.
- **500 Internal Server Error**: Check the PHP error logs in cPanel or make sure your PHP version is 8.1+.
- **Database Connection Error**: Double-check your `.env` credentials.

Your backend is now ready! You can test it by visiting `https://yourdomain.com/api{"error":"Endpoint not found"}`.
